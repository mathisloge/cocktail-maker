== Context

We need a robust relational database to store our transactional data. The data has a highly structured schema and requires ACID compliance.

== Decision

We will use PostgreSQL as our primary datastore.

== Consequences

* **Good:** High reliability and standard SQL support.
* **Good:** Free and open source.
* **Bad:** Requires specialized tuning for high-concurrency write loads.
