= Architecture Decisions

Only the decisions themselves are stated here. Their context, alternatives, and consequences are
recorded in the referenced ADRs.

* *Station-Pod Communication Model:* request-response with the Station as the sole initiator, decided
  at the system level in xref:system:adr:003-protocol-communication.adoc[ADR-003].
* *Physical Transport:* USB CDC, decided at the system level in
  xref:system:adr:002-pysical-pod-connection.adoc[ADR-002].
* *Protocol Codegen:* the shared, code-generated schema described in xref:system::protocol.adoc[],
  decided at the system level in xref:system:adr:006-protocol-serialization.adoc[ADR-006].
* *UI Toolkit:* Slint, decided in xref:adr:001-ui-toolkit.adoc[ADR-001].
* *Source Organization & Compilation Model:* {cpp}26 modules, decided in
  xref:adr:002-cpp-module-compilation-model.adoc[ADR-002].
* *Asynchronous Execution Model:* Boost.Asio and Boost.Cobalt coroutines on a single shared executor,
  decided in xref:adr:003-asynchronous-execution-model.adoc[ADR-003].

The communication model, transport, and protocol codegen are shared with the Pod and decided once at
the system level (xref:system::system-design.adoc[]). The UI toolkit, source organization, and
asynchronous execution model are choices internal to the Station.
